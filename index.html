QCM CSS – Révision complète

1. Quel sélecteur cible un identifiant ?
A. .menu
B. #menu
C. menu
D. menu#id
Réponse : B. #menu
2. Quelle propriété enlève le soulignement d’un lien ?
A. text-align
B. underline: none
C. text-decoration: none
D. decoration-remove
Réponse : C. text-decoration: none
3. Quelle propriété place les éléments horizontalement en Flexbox ?
A. flex-direction: column
B. flex-order
C. justify-content
D. flex-direction: row
Réponse : D. flex-direction: row
4. Que fait display: inline-block ?
A. Prend toute la ligne
B. Ne peut pas avoir de width
C. Reste en ligne et peut avoir une taille
D. S’affiche uniquement en colonne
Réponse : C
5. Quelle propriété arrondit les coins ?
A. border-rounding
B. border-radius
C. radius-corner
D. border-curve
Réponse : B. border-radius
6. Quel code active une media query pour smartphone (max 768px) ?
A. @phone (768)
B. @media (max-width: 768px)
C. media-query smartphone()
D. @smartphone width 768
Réponse : B
7. justify-content centre les éléments sur…
A. l’axe secondaire
B. l’axe principal
C. la largeur totale
D. le texte
Réponse : B. l’axe principal
8. Quelle unité dépend de la racine html ?
A. px
B. em
C. rem
D. %
Réponse : C. rem
9. Quelle propriété enlève les puces d’une liste ?
A. dot:none
B. no-bullet
C. list-style-type: none
D. remove-style
Réponse : C. list-style-type: none
10. Quelle propriété aligne verticalement (en row) en Flexbox ?
A. align-items
B. justify-content
C. vertical-align
D. align-content
Réponse : A. align-items
